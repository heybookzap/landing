## 4. 카피라이팅 원칙 요약

### 단어 선택 기준

| 피할 단어 | 대체 단어 | 이유 |
|---|---|---|
| "결제하세요" | "자산으로 바꾸십시오" | 비용이 아닌 투자 프레임 |
| "구독하세요" | "전환하십시오" | 능동적 결정 강조 |
| "삭제됩니다" | "소멸됩니다" / "사라집니다" | 비가역적 손실감 극대화 |
| "업그레이드" | "보존하십시오" / "예치하십시오" | 방어 본능 자극 |
| "무료 → 유료" | "체험 → 자산화" | 가치 전환 프레임 유지 |

### 프레임 우선순위

1. **손실 회피 (Loss Aversion)** — "버린다"는 공포가 "얻는다"는 기대보다 항상 먼저
2. **복리 자산화 (Compounding Frame)** — 노력을 금액으로 환산해 구체성 부여
3. **시간 압박 (Urgency)** — 남은 시간을 항상 숫자로, 동사를 현재형으로
4. **엘리트 정체성 (Identity)** — "리더", "자산", "엘리트 시스템"으로 자아 투영

### 문장 길이 기준

- **푸시 알림:** 문장 당 15자 이내
- **팝업 헤드카피:** 한 줄 14자 이내 · 2줄 구성
- **이메일 본문:** 단락 당 3문장 이내 · 빈 줄로 호흡 확보
- **버튼 문구:** 10자 이내 (최종 확인 버튼은 의도적으로 길게)
